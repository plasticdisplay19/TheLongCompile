---

## 🕹️ How to Play

### Controls
Use your keyboard's arrow keys to maneuver the tiles:
- ⬆️ **Up Arrow:** Slide all tiles up.
- ⬇️ **Down Arrow:** Slide all tiles down.
- ⬅️ **Left Arrow:** Slide all tiles left.
- ➡️ **Right Arrow:** Slide all tiles right.
